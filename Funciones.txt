Funciones Personalizadas 
- es_numero: Valida que las entradas sean numéricas, permitiendo que el programa maneje decimales sin cerrarse por errores.
- promedio / nota_mas_alta / nota_mas_baja: Realizan cálculos estadísticos sobre colecciones de datos numéricos.

- contar_frecuencia: Analiza una lista de palabras y genera un diccionario donde la clave es la palabra y el valor es la cantidad de veces que aparece.

Métodos, Atributos y Control de Flujo
- Manejo de Diccionarios (items, values, get): Permiten acceder a las claves, a los datos, o realizar búsquedas seguras dentro de un diccionario.
- Manipulación de Cadenas (split, lower, isdigit): Herramientas para dividir textos, normalizarlos a minúsculas o verificar si son dígitos antes de procesarlos.
- Gestión de Listas (append): Método para añadir nuevos elementos dinámicamente a una lista de datos.
- Ciclo while: Crea bucles de ejecución continua para programas interactivos con menús.
- Ciclo for: Estructura principal para recorrer (iterar) datos y aplicar lógica individual a cada elemento.
- Condicionales (if / elif / else): Bloques de decisión que controlan qué parte del código se ejecuta según las entradas del usuario o los resultados de las operaciones.

9. 

Funciones Personalizadas
- agrupar_por_categoria: Organiza una lista de tuplas en un diccionario donde cada clave es una categoría y su valor es una lista con los nombres de los productos correspondientes.

Métodos y Control de Flujo
- append(): Agrega cada producto a la lista específica de su categoría dentro del diccionario.
- items(): Permite iterar sobre las categorías y sus respectivas listas de productos para la impresión final.
- join(): Concatena los elementos de la lista de productos en una sola cadena de texto separada por comas.
- Bucle for: Recorre tanto la lista inicial de tuplas para agruparlas como el diccionario resultante para mostrar los datos.
- Condicional if not in: Verifica si la categoría ya existe como clave en el diccionario; si no, la crea con una lista vacía para evitar errores al agrupar.
